하우스 오브 옹알 파일 서비스 API 문서입니다.

파일(User) API

파일은 miniO의 Object Storage에 저장됩니다.

파일 서버의 API는 보통 내부 service 간의 API, message를 통해 실행됩니다.

파일 업로드 API

파일을 업로드합니다.

HTTP request

POST /files/upload HTTP/1.1
Content-Type: multipart/form-data;charset=UTF-8; boundary=6o2knFse3p53ty9dmcQvWAIx1zInP11uCfbm
Host: api.houseofongr.com

--6o2knFse3p53ty9dmcQvWAIx1zInP11uCfbm
Content-Disposition: form-data; name=metadata
Content-Type: application/json

{
  "domain": "universe",
  "ownerID": "019812d4-73e3-7d80-9e8e-aa62af94c0b3",
  "accessLevel": "PUBLIC"
}

--6o2knFse3p53ty9dmcQvWAIx1zInP11uCfbm
Content-Disposition: form-data; name=file; filename=image.png
Content-Type: image/png

<<png data 1>>
--6o2knFse3p53ty9dmcQvWAIx1zInP11uCfbm--

Request parts

Part Description

metadata

업로드할 파일의 정보를 포함하는 Json 문자열입니다.

file

업로드할 파일입니다.

Response body

{
  "id" : "01983035-62fb-7bce-89e6-e028cc166576",
  "fileUrl" : "http://localhost:9000/universe/image/01983035-62fb-7bce-89e6-e028cc166576.png",
  "uploadedTimestamp" : 1753155461
}

Response fields

Path Type Description

id

String

저장된 파일의 ID입니다.

fileUrl

String

저장된 파일의 url 경로입니다.

uploadedTimestamp

Number

파일이 생성된 시간입니다.

에러코드

해당 서비스에서 발생 가능한 에러코드는 다음 표에서 확인할 수 있습니다.

해당 표에서 확인되지 않는 에러코드는 별도 문의 바랍니다.

도메인 계층 에러코드

Code 설명(메시지) HTTP 상태 코드 코드 원인
